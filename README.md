1. What is difference between var, let and const?
Ans: var, let and const are used to declare variables in javascript. var works in function scope, while let and const work in block scope. The value of let can be changed, but the value of const are also hoisted but remain in the Temporal Dead Zone. So, they cannot be used before their declaration. 

2. What is spread operator (...)?
Ans: The spread operator (...) is used to expand or copy elements of an array or object into another array of object.

3. What is the difference between map(), filter() and forEach()?
Ans: map() creates a new array by transforming each element, filter() creates a new array based on a condition and forEach() only loops through elements without returning a new array.

4. What is an arrow function?
Ans: An arrow function is a shorter and mordern way to write functions in javascript using the => syntax.

5. What are template literals?
Ans: Template literals allow you to create strings using backticks and easily insert variables or expressions inside a string.
